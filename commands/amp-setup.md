# /amp:setup

Check Amp CLI installation status and configure the plugin.

## Procedure

1. Check installation with `amp --version`
2. Check auth status with `amp auth status`
3. If not installed: guide `npm install -g @sourcegraph/amp@latest`
4. If not authenticated: guide `amp auth login`
5. Ask whether to enable the review gate
